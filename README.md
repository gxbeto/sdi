# sdi
Sistema Distribuido de Inventario (SDI) — API REST del módulo Stock construida con FastAPI y PostgreSQL. Gestión de productos, reservas y ventas con comprobantes verificables, procesamiento idempotente de compras vía RabbitMQ, migraciones Alembic y contrato OpenAPI 3.2 documentado. Incluye simulador de operaciones y despliegue automatizado.
